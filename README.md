# ご飯いこ！

## サービス概要
“ご飯いこ！”と言ったその瞬間から、スムーズに決められるアプリ。  
言いにくいことも共有しながら、”スムーズ”かつ”一人一人の意見を反映”したご飯場所を決めるサポートツールを提供する。

## このサービスへの思い・作りたい理由
友人・同僚など、ご飯に行く機会は定期的にある。  
ご飯に行く際には「行き先」、予約もする場合は「予約の時間」「コース」など全てが決定するまでにいくつかのやりとりが発生する。  
そのやりとりが普段のコミュニケーションツールであるLINE上でだと、「候補先の情報がストックづらい（流れてしまう）」、「投票で決める際に、一票しか投票できない」など若干やりづらさを感じていたため、飲食店決めのサポートツールがあれば便利ではないかと考えた。  
また、ご飯に行くのは気心の知れたなんでも言い合える相手ばかりではない。  
場合によっては、相手に少し伝えにくい「金額帯はこのぐらいがいいなあ」「ちょっとこの食べ物苦手だけど。。」ということも存在する。  
上記の理由から、飲食店決めにスムーズかつ、伝えにくいことも一緒に共有できるコミュニケーションのサポートツールを作成したいと考えた。

## ユーザー層について
- 円滑にグループで食事先を決めたい方
- 普段の食事の行き先決めで周りに合わせがちの方

## サービスの利用イメージ
 
### ご飯の約束が確定するまでのステップ
1. お出かけグループのリンクを作成
2. リンクをグループに共有
3. 食事先の候補登録（期限も設定できる）
4. 同時に苦手な食べ物・想定している価格帯・想定開始時間などを記入
5. 登録したお出かけ先に対して、1~5段階のポイントで投票（投票期限も設定可能）
6. 得点を集計して、ポイント数第1位〜3位のお出かけ先を表示
7. 価格帯・開始時間などはそれぞれの意見を表示、価格は中間の値も表示


### その他
- 行ったところは、「行ったリスト」に、候補には上がったが行かなかったところは「行っていない」リストに入る


## ユーザーの獲得について
Xでの発信や、友人間でシェアされることで、ユーザーの獲得をしていきたい。


## サービスの差別化ポイント・推しポイント
### LINEのみで行き先を決める際との差別化ポイント
- アイデアをストックしてグループ間で共有できる点  
本サービスを利用することで食事先の候補を各々が登録でき、候補も一覧化されるため見やすくなる。
- 投票機能  
投票機能に1~5のポイントをつけることで、行きたい度合いを表現可能。
　また、最終的なポイント数のみ表示する（リアルタイムで得点を表示しない）ことで、周囲の意見を気にすることなく投票することができる。
- 必要事項のやり取り短縮  
各自が質問内容に答えるだけで、予約時間などの各自の都合を反映すべきもののやり取りを短縮できる。
- 匿名で言いづらさを解消   
価格帯や苦手な食べ物など、周りに伝えにくい場合があるが、本サービスでは匿名で記入できて、意思の共有ができる。

### アンケート系のアプリとの差別化ポイント
- 食事先を決めるまでに必要なアンケートがデフォルトでセットされているので、アンケートから回答までがスムーズになる。

## 機能候補
### MVPリリース時
- メンバー招待（リンクの共有）
- 食事先候補登録
- 投票
- ランキング
- 食事先以外の匿名投票（集合時間、価格帯、苦手なもの）

### 本リリース時
- 投票締切時間の設定
- 行ったリスト・行ってないリスト
- 周辺のおすすめスポット紹介
- LINEでの通知機能（候補締切通知・投票締切通知・行き先決定）
- 任意の質問追加機能
- 場所の表示機能（マップ上に候補の場所を表示）
